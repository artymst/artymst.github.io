Overview:

PhysicsGraphPyTM is a beginner-friendly Python library for simulating and visualizing motion in physics. With this library, you can easily create and analyze kinematics and projectile motion scenarios, generate position, velocity, and acceleration graphs, as well as animate motion using simple code. PhysicsGraphPy is ideal for students, educators, and anyone exploring basic physics concepts programmatically. This package is distributed under the MIT License and is available for manual installation directly from this website. Just download the installation files and follow the provided instructions to start experimenting with motion graphs and physics simulations in your own Python projects.

Features:

PhysicsGraphPy offers a rich set of features for exploring motion in physics. It supports simulation of free-fall, kinematics, and projectile trajectories, allowing you to analyze movement with custom parameters. The library can generate easy-to-read position, velocity, and acceleration graphs, and also provides animation tools to visualize motion step by step. With built-in calculators for quantities like maximum height, time of flight, and range, as well as options to export data in CSV or Markdown format, PhysicsGraphPy makes it simple to study and present physics concepts. For added flexibility, the package includes unit conversion tools, SI unit checking, and options to customize graph styles—all designed to streamline physics experiments and learning in Python.

Requirements:

To use PhysicsGraphPy, you need to have Python 3.7 or higher installed on your computer. The library also requires the popular matplotlib and numpy packages for graphing and numerical calculations. Before installing PhysicsGraphPy, make sure these dependencies are available, or simply use pip to handle the installation automatically. No other specialized software is required—just a standard Python environment is enough to start simulating and visualizing motion in physics with this package.

Supported systems:

PhysicsGraphPy is designed to work on all major operating systems that support Python, including Windows, macOS, and most Linux distributions. As long as you have a compatible version of Python installed, along with the necessary dependencies, you can use PhysicsGraphPy on desktop or laptop computers. The library is intended for standard desktop Python environments and may not be fully compatible with limited mobile or browser-based interpreters.

Main Commands:

PhysicsGraphPy offers a set of intuitive commands to make simulating and plotting motion in physics easy for beginners and advanced users alike. Below are the main functions and classes you’ll use when working with this library:

Commands

MotionGraph - Create and simulate basic motion. 
obj = MotionGraph(10, 0, -9.8)  # start at 10 m, 0 m/s, gravity
obj.simulate(3)

ProjectileMotion - Model and analyze projectile trajectories. 
pm = ProjectileMotion(20, 45)  # 20 m/s at 45 degrees
pm.simulate(pm.time_of_flight())

plot_all – Plot position, velocity, and acceleration graphs 
 plot_all(obj)

plot_projectile – Plot the path of a projectile 
 plot_projectile(pm) 

 plot_projectile(pm)

animate_motion – Animate how an object moves 
 animate_motion(obj)

export_csv – Export simulation data to a CSV file 
 export_csv("free_fall.csv", obj.times, obj.positions, obj.velocities)

convert_units – Convert between common units 
 meters = convert_units(100, "cm", "m") # returns: 1.0

format_markdown – Convert results to Markdown format for reports 
 summary = format_markdown("Results", {"Max Height": 10, "Range": 40})

check_si_units – Check if a unit is a standard SI unit 
 check_si_units("m", "m") # No error if SI
